# Professional Portfolio - Hakizimana Sylvan

A modern, responsive personal portfolio website built with HTML5, CSS3, and Vanilla JavaScript. Features a neural network-inspired interactive background, glassmorphism design, and scroll reveal animations.

## 🚀 Live Demo

[Link to GitHub Pages](https://github.com/username/repo-name)

## 🛠️ Built With

- **HTML5**: Semantic structure and accessibility.
- **CSS3**: Custom properties, Flexbox/Grid, and Animations.
- **JavaScript (ES6+)**: Canvas animations and Intersection Observer API.
- **FontAwesome**: Scalable vector icons.
- **Google Fonts**: 'Inter' and 'Orbitron' typography.

## 📂 Project Structure

```
/
├── index.html      # Main entry point
├── style.css       # Global styles and responsive design
├── script.js       # Particle animation and interaction logic
└── README.md       # Project documentation
```

## 📦 Deployment to GitHub Pages

1.  Push this repository to GitHub.
2.  Go to **Settings** > **Pages**.
3.  Under **Source**, select `Deploy from a branch`.
4.  Select `main` (or `master`) branch and `/root` folder.
5.  Save and wait for the site to build.
